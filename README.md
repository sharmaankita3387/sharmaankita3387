## Featured Project: Spam Email Classifier (From Scratch in C)
**Repository**: [SpamEmailClassifier](https://github.com/sharmaankita3387/SpamEmailClassifier) </br> </br>
**Description**: A lightweight, dependency-free spam email classifier implemented entirely in C, using a mathematically rigorous Naive Bayes model. </br>

**Goal**: `Most ML projects hide the fundamentals behind libraries. I wanted to understand ML at the lowest level: probability, memory, performance, and correctness by implementing everything from scratch.`

## **What I Built**
1. Core ML Implementation
- Implemented Naive Bayes text classification from first principles
- Applied Laplace smoothing to prevent zero-probability failures
- Used log-probabilities to avoid numerical underflow
- Designed probabilistic outputs with configurable classification thresholds

2. Systems & Software Engineering
- Pure C implementation with zero dependencies, portable across platforms
- Memory-efficient design supporting 50,000+ word vocabularies
- Thread-safe architecture, suitable for server-side integration
- Clean modular structure (classifier_core, probability_calc, naive_bayes)

3. Testing & Reliability
- Comprehensive test suite with 95%+ code coverage (gcov)
- Deterministic behavior and reproducible results
- Defensive memory management and clean resource teardown </br>
